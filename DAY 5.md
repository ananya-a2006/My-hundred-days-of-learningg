🐍 Day 5 – Functions

🔹 Defining a Function

**Syntax:**
```python
def function_name():
    # code block
```

🔹 Function with Arguments

```python
def greet(name):
    print("Hello,", name)

greet("Ananya")
```

🔹 Returning Values

```python
def add(a, b):
    return a + b

result = add(5, 3)
print("Sum:", result)
```

✅ Example – Function with Arguments and Return
```python
def calculate_area(length, width):
    area = length * width
    return area

print("Area:", calculate_area(5, 3))
```
